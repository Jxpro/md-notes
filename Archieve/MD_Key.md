## 常用快捷键

|       功能       |      快捷键       |
| :--------------: | :---------------: |
|     插入代码     | Ctrl + Shift +  ` |
|    插入代码块    |     Ctrl + `      |
|     打开文件     |     Ctrl + O      |
|   打开文件夹…    |  Ctrl + Alt + O   |
|    选中当前词    |     Ctrl + D      |
|  选中当前行或句  |      Alt + L      |
| 选中当前格式文本 |     Ctrl + E      |
|  删除当前行或句  |     Ctrl + L      |
|    水平分割线    |     Ctrl + -      |
|     删除样式     |     Ctrl +  \     |
